# CausalGEP
CausalGEP
CausalGEP (Causal Gene-Environment-Phenotype) is an innovative causal machine learning framework designed specifically for analyzing the dynamic interaction relationships between genes, environment, and phenotypes in complex biological systems. This framework integrates multimodal deep learning, structural causal models, and counterfactual reasoning technologies, breaking through the limitations of statistical correlation in traditional genome-wide association studies (GWAS) and achieving a paradigm shift from "correlation discovery" to "causal inference".
